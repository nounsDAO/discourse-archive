First of all, I am not very good at English, so please understand. Let’s get straight to the point.
In this Nouns Proposals, I found that all proposals that have been approved are paid in a lump sum. Suppose, a very bad idea happens… For example: the project is stopped or even RUGged. then such a loss must also be a loss for all voters.
Therefore, I suggest that we initiate a proposal to add a Superfluid option. Projects that have been approved report progress as required, and if the deadline is exceeded, we immediately vote to stop paying the balance so that unnecessary losses can be mitigated. Although my idea is a bit cruel for the project side. But I would prefer Nouns to be healthy and spread to every corner.
