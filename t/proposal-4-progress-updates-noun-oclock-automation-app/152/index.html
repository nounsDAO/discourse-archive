An update on our Nouns automation apps

  • You can now build your own automation with no-code using apps we built for Zapier & Integromat.
  • These apps are now working with two triggers: Auction Settled & New Proposal.
  • For example, you can create an automation that updates you by email/text when a new auction or proposal are created. We also created a Twitter bot and Google Calendar to showcase what’s possible.

Please play around with it and let us know what you like or what’s missing

Links to the apps & a few implementation templates are here: https://www.verbs.team/nouns/automations

verb-d

We have just added another trigger to the Zapier & Integromat apps:

X Time Before Auction Ends

  • This means you can create automations to get notified X time before the auction ends. You can, for example, set up an SMS or email to be sent so you will never miss Noun O’clock.
  • You will see examples of this automation implemented on https://twitter.com/NounsCalendar (10 minutes and just before Noun O’clock)
  • Let us know if you would anything different with these!

Hi all,

A (final) update on our automation project:

  • Triggers: You can create automations on Zapier or Integrmat that are triggered when:
    • Triggers when a Nouns auction is settled and a new Noun is created
    • Triggers when a new NounsDAO proposal is submitted on-chain
    • Triggers X time before auction ends
  • Links to the apps:
    • Zapier: https://zapier.com/developer/public-invite/145904/8a8e34e04debec63de5fef1526fe3879/
    • Integromat: https://www.integromat.com/en/apps/invite/6b5f6756542eb1dc5aac97ff95cf2e68
  • Open source: We have open sourced the code for these apps, feel free to open issues/pull requests or fork:
    • https://github.com/verbsteam/nouns-automations
  • For guides on how to implement your own automations, check out our documentation:
    • https://verbs.notion.site/Nouns-Automations-using-Integromat-Zapier-7f3af840dc1d4a04b1d728f978c785b0

Live implementations, built using the Integromat no-code app!

  • NounsCalendar - subscribe to see when NounO’clock is set to happen tomorrow (also cool to see how it’s evolving over time), as well as when voting on proposals starts and ends: https://nouns.notion.site/Nouns-Google-Calendar-c8314d548aed4aa1b841cb868b9ea0a8
  • NounsCalendar Twitter Bot - follow to get notified when it’s NounO’clock, when a new Noun is born, and when a new proposal is submitted on-chain: https://twitter.com/nounscalendar

We loved building this and hope the community will use this infrastructure to build more cool stuff!

If you have any feedback or requests, find me on discord or twitter

Thanks
